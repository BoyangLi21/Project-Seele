# EVA 电力与脐带供电人工测试

该系统把 EVA 的“能动”与服务端权威电力绑定。供电柱范围内由外部电网维持，断缆后使用五分钟内部电池；耗尽后机体停机，但驾驶员始终可以正常弹出。

## 当前数值（新增，待玩家拍板）

| 项目 | 默认值 | 配置键 |
|---|---:|---|
| 内置电池 | 6000 tick（5 分钟） | `eva_power.capacityTicks` |
| 脐带连接距离 | 32 格 | `eva_power.umbilicalRange` |
| 接电维修 | 每秒 1.0 生命值（半颗心） | `eva_power.repairPerSecond` |

上述数值没有改变任何武器、使徒、A.T. 力场耐久或 EVA 最大生命值。

## 单人快速验收

1. 生成或找到一台 EVA，并执行 `/give @s projectseele:umbilical_pylon`。在机体 32 格内放置供电柱。
2. 从背部插入栓入口登乘。左上角必须显示“外部供电 / 脐带已连接”，电池条满载；第三人称可看到从供电柱到 EVA 背部下垂的橙色脐带。
3. 让 EVA 受少量伤并保持接电，生命值应每秒恢复 1.0。使徒和附近其他 EVA 不得被同一供电柱隔空治疗，只有各自处于连接范围内的 EVA 生效。
4. 打掉供电柱。HUD 立即切换到内部电池倒计时，初始接近 `5:00`；走、跑、蹲、卧倒、跳、枪械、粒子刀、朗基努斯和 A.T. 力场仍可使用。
5. 为加速测试，对最近机体执行 `/data merge entity @e[type=projectseele:eva_unit01,limit=1,sort=nearest] {SeelePowerTicks:80}`。约四秒后必须出现停机警告。
6. 电力为零时确认：移动、跳跃、武器切换、近战、射击、蓄力和展开 A.T. 力场全部无效；已有力场自动关闭，机体速度归零。
7. 按原退出键弹出插入栓。断电不能把驾驶员困在机体内。
8. 重新在 32 格内放置供电柱。电量恢复满载、控制恢复、接电维修重新生效。
9. 把供电柱移到 33 格外；最迟一秒后必须断开。来回跨越边界时不能每 tick 刷屏或复制电缆。

## 重启与多人验收

1. 断缆并消耗一部分电量，正常保存退出，再进入同一世界；剩余 `SeelePowerTicks` 必须保持，不能回满。
2. 两名玩家分别驾驶两台 EVA，在同一供电柱 32 格内移动；两台机体各自显示相同外部电网状态，但电量和伤势仍独立同步。
3. 一名玩家留在 NERV Operations。驾驶员屏与战略文字屏的 `POWER UMBILICAL/INTERNAL` 和剩余值必须与驾驶 HUD 一致。
4. 驾驶员断线时不继续消耗内部电池；重新上线登乘后从存档值继续。

## 配置测试

可在 `config/projectseele-common.toml` 调整 `capacityTicks`、`umbilicalRange` 和 `repairPerSecond`。为了快速反复测试，可暂时把 `capacityTicks` 改成 `200`（10 秒），验收后恢复 6000。

## 离线门禁

```bat
python tools\validate_eva_power_contract.py
```

离线门禁检查注册、同步/存档、有限加载索引、服务端电力循环、统一控制锁、HUD/指挥室、电缆和资源文件。它不能替代玩家对五分钟压力、接电范围和电缆观感的实机判断。
## 连续发射路线验收

1. 执行 `/seele geofront setup`（旧地图可直接执行 `/seele geofront link` 触发有界修复）。三条路线各自应有 1 个地下供电桩和 1 个地表供电桩，共 6 个。
2. EVA 停在地下床位时，供电桩距床位约 10 格；登乘后 HUD 和指挥室都应显示 `POWER UMBILICAL`，橙色脐带连接到机体背部。
3. 发射载台沿真实 522 格竖井上升。离地下供电桩超过 `umbilicalRange`（默认 32 格）后，脐带必须断开，HUD 切换为约 `5:00` 的内置电池倒计时；断缆不应中止发射。
4. 到达地表后，靠近同路线地表供电桩（床位东侧约 11 格）应自动重新接电、恢复满电并重新显示电缆。
5. 破坏任意一个路线供电桩后执行 `/seele geofront sortie_audit`，审计必须失败；重新执行 `/seele geofront link` 后应补回缺失供电桩并恢复通过。
