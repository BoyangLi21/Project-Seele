# EVA 武器与战略爆炸测试

本页记录 `recovery/visual-foundation` 当前可复现的武器验收流程。请只在可丢弃的测试世界里验证阳电子炮和 N²；两者会永久改写地形。

## 共通操作

- `R`：依次切换徒手 → 粒子刀 → 阳电子炮 → 朗基努斯枪/二号机专武 → 帕雷特冲锋枪 → N² 自我毁灭系统。
- `左键`：近战武器攻击；冲锋枪模式下按住连续射击。
- `右键`：阳电子炮蓄能；N² 模式下持续解锁，提前松开立即中止。
- 所有枪械射线、第三人称枪口俯仰和第一人称光学轴共享同一组受限瞄准数据。

## 阳电子炮

1. 切到“阳电子狙击炮”。第一人称不应再看到接收机、双臂或炮管塞满屏幕；炮体仍应在第三人称存在。
2. 按住右键。应进入原创“屋岛作战 / 光学射击管制”界面：两枚捕获符号逐步向中心重合，测距与目标状态实时更新，视场随蓄能收窄。
3. 鼠标向上时，第三人称炮口应向上；鼠标向下时应向下。镜头、炮口和最终命中不得反向。
4. 蓄满后界面转绿；松开右键发射。未蓄满松开只取消。
5. 命中地形后，伤害立即结算；默认半径 64、深 32 格的弹坑按每 tick 2048 个方块的全局预算向外展开。它不会用一次同步爆炸冻结服务器。
6. 人类武器不生成使徒十字光柱。阳电子炮只生成核闪光、火球和冲击环。

## 帕雷特冲锋枪

1. 切到“EVA 帕雷特冲锋枪”。本机安装来源完整的 TV Pallet Rifle 时显示本地审阅的 292 面模型；否则显示 Project SEELE 原创 240 面备用模型。两种资源都使用同一个双手瞄准与枪口 socket。TV 模型仅限本机测试，发布前必须取得原作者许可。
2. 按住左键。默认每 3 tick 发射一次、单发 22 伤害、射程 192 格。
3. 每发只出现短促曳光和命中火花；地面不得出现爆炸、弹坑或核爆效果。
4. 完整 A.T. 力场必须将子弹伤害归零，也不得被子弹削减。先用 EVA 近战打掉力场后，冲锋枪才能伤害使徒本体；拉米尔暴露核心时也可受普通枪伤。

## N² 自我毁灭系统

1. 切到“N² 自我毁灭系统”，确认 EVA 以双手在胸前持有可见的装甲炸弹，再按住右键 5 秒。红色界面显示倒计时；中途松手应完全取消。
2. 倒计时结束后驾驶员和当前 EVA 必须死亡，爆心出现无十字的超级核爆。
3. 默认效果严格使用阳电子炮的三倍半径、三倍深度、三倍即时伤害和三倍客户端特效比例：地形半径 192、深度 96、即时伤害半径 108、中心伤害 480。
4. 为避免偷偷生成上千区块，弹坑每次处理方块前只访问当时仍已加载的区块，不主动加载新区块；地形改写继续受每 tick 2048 方块的全局预算限制。

## 配置

对应 `config/projectseele-common.toml`：

- `eva_pallet_smg.damage/range/intervalTicks`
- `positron_cannon.explosionRadius/blastDamage/terrainRadius/craterDepth`
- `strategic_explosions.n2ArmTicks/blocksPerTick`

结构回归：`python tools/validate_weapon_systems.py`。完整本地资源与动作矩阵仍由桌面启动器在启动前 fail-closed 验证。
