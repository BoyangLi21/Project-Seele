## §1 愿景与设计支柱

1. **名场面还原优先**：屋岛作战（雅茹狙击）、初号机暴走、可升降的第三新东京市、量产机之战、第三次冲击。每个 Phase 至少完整还原一个名场面。
2. **玩法闭环**：使徒来袭事件 → 驾驶 EVA 迎击 → 核心/材料 → 科技与同步率成长 → 剧情推进 → 终局抉择。
3. **开源协作友好**：模块化注册、资产可热替换（社区模型/贴图直接以资源包覆盖）、文档齐全。
4. **合规**：khara 二创指引框架内，永久免费，零官方素材。

**剧情主线（以进度树 advancement 承载）**：
`序·警报初响`（击杀首个使徒）→ `破·初号机启动`（同步率系统）→ `急·使徒图鉴`（Sachiel/Shamshel/Zeruel...）→ `SEELE 剧本`（朗基努斯 + 量产机）→ `终·第三次冲击`（生命之树，玩家抉择）→ `后日谈·红之海`。

### 6.2 第三次冲击事件（ThirdImpactEvent）

**架构**：`SavedData` 状态机（全服唯一，阶段推进按 tick 驱动），S2C 事件包广播每个阶段，所有客户端特效本地渲染。**不可逆，触发前弹确认 UI。**

**触发**：完成「SEELE 剧本」进度链后，在莉莉丝面前使用朗基努斯枪。

**分镜（游戏内时间轴）**：

| 时刻 | 阶段 | 实现 |
|---|---|---|
| T+0s | **天空剧变** | `DimensionSpecialEffects` 覆盖：天空渐变橙红、云层消失、地平线出现巨大光环；全服 Title「第三次冲击」+ 低频轰鸣（原创音效） |
| T+20s | **生命之树显现** | `RenderLevelStageEvent(AFTER_SKY)` 天空渲染：**卡巴拉生命树**——10 个源质光球（Sephirot）+ 22 条路径连线，发光线框几何（复用直连渲染技术，八面体的放大版思路），整体缓慢自旋+脉动；树心为光之巨人剪影（billboard）。这是全 mod 视觉巅峰，3070 开发机可上 shader 调试 |
| T+60s | **反 AT 力场波** | 以莉莉丝为中心每 8s 扩张一波（半径 +48，上限 1024）：波前为可见光环（地表扫过的橙色波纹）；波及的 LivingEntity（未受保护者）→ **LCL 化**：十字光爆 + 实体消失 + 原地生成 LCL 液体池。实体扫描按波前环带分片处理防卡顿 |
| T+180s | **玩家抉择** | 每位玩家弹出 GUI：**「回归补完」**→ 旁观者模式 + 橙色滤镜 + 置于 LCL 之海上空漂浮（可随时反悔"从海里爬回来"复活）／**「拒绝补完」**→ 存活，进入 EoE 世界状态 |
| 收尾 | **EoE 世界** | 持久化标记：海洋渲染为红色（生物群系覆盖/液体 tint）、天空残留光环、月亮贴图替换（血痕）、玩家最近海岸生成**红之海滩**结构（两把插在沙中的断剑彩蛋可选）。进度：「在海边醒来」 |

**多人语义**：世界状态全局唯一，补完抉择 per-player。一人补完一人拒绝时各自视角正确分歧。

**技术风险**：天空渲染兼容性（OptiFine/Iris 用户后处理冲突→client config 提供降级开关）；波前性能（分片+限流）；LCL 化的掉落物洪水（合并物品实体）。

**验收**：单人+局域网双人完整触发三冲，两种抉择路径正确，退出重进存档后 EoE 状态保持。
**估时**：4–6 周。
